2) В этом пректе реализуется сервис, который принимает и отвечает на HTTP запросы

3) Для работы проекта нужно: а) импортировать файлы из репозитория 'MediaTest' себе на ПК б) запустить MySQL через MAMP или openserver (данные от MySQL в пункте 5) в) запусить сервер, 
   выполнив консольную команду 'py manage.py runserver', после чего перейти по выданной ссылке на local.server

4)
	a)Для промотра всем городов в БД, в адресную строку нужно вписать следующий адресс: http://127.0.0.1:8000/city 
	b)Для промотра всем улиц в БД, в адресную строку нужно вписать следующий адресс: http://127.0.0.1:8000/city//street/<id выбранного города> 
	c)Что бы добавить магазин в БД, отправим в сервис POST запрос, для этого нужно установить Google расширение - Postman, вставить в него адресс http://127.0.0.1:8000/shop/, 
	  выбрать вид запроса POST, выбрать пункт "body", вписать данные магазина, который хотим создать, и нажать кнопку SEND 
	d)Для промотра всех магазинов на определенной улице определенного города, в адресную строку нужно вписать следующий адресс: http://127.0.0.1:8000/shop/?street=<id выбранной улицы>&city=<id выбранного города>&open=0/1


5) Данные от БД:

        'PASSWORD': 'root'

        'PORT': '3306'
